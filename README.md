# Solavisetech-women-in-data-cohort2


### About the Program
The Solavise Tech Women in Data Program is a 2-month intensive 
learning experience for women building careers in data and technology. 
Cohort 2 runs from May to June 2026, covering Python for data analysis, 
data visualization, GitHub, and portfolio building — fully online and community-driven.

### About Me
I'm Lindsey-Johanne, a public health professional from Cameroon working 
at the intersection of digital health, health informatics, and data. 
My background spans research, monitoring & evaluation, and the design and evaluation of digital health interventions.
I'm currently building 
my technical data skills to complement my public health foundation.

### My Expectations
I expect this program to be both challenging and practical. Beyond the technical curriculum, I'm looking forward to learning alongside a community of driven women who push each other forward. Iron sharpens iron — and I expect to leave this cohort sharper than I came in.

### What I Hope to Learn
- Deepening my Python skills for practical data analysis
- Data visualization and storytelling with data
- GitHub for version control and portfolio building
- How to think and work like a data professional
- How to collaborate, build, and grow within a community of women in tech
